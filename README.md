                                               OCR-Автоматизация ввода данных

Важное примечание: Код, комментарии и имена функций (sozdat_text, sozdat_text_simple и др.) написаны на русском языке для максимального соответствия реальной рабочей среде и операционным задачам в русскоязычной IT-инфраструктуре.

                                                        Цель проекта:
Микро-скрипт на Python для быстрой автоматизации рутинного ввода данных из скриншотов и изображений. Разработан для решения специфической задачи: конвертации длинных списков (номеров терминалов, VIN, данных мониторинга) со скриншотов в редактируемый текстовый формат.
Этот инструмент сокращает время, затрачиваемое на перепечатывание данных, минимизирует ошибки, повышая эффективность работы хелп-деска.

                                                 Функционал и технические решения
Скрипт содержит несколько функций, демонстрирующих подход к надежному распознаванию текста (Optical Character Recognition, OCR).

1. Основной метод (sozdat_text)

Предобработка OpenCV: Для повышения точности распознавания применяется комплексная предобработка изображения:
Проверка размера и автоматическое масштабирование мелких изображений.
Конвертация в оттенки серого.
Увеличение контраста (cv2.equalizeHist) для выделения текста.
Удаление шума (cv2.medianBlur).
Гибкое OCR: Используется easyocr.Reader(['ru', 'en']) с поддержкой русского и английского языков.
Обработка ошибок: Встроенные try...except блоки для безопасной работы с файлами.

2. Вспомогательные методы
sozdat_text_simple: Базовый метод без предобработки для быстрой проверки.
sozdat_text_robust: Попытка использовать несколько методов распознавания в случае сбоя, что повышает отказоустойчивость.

Используемые технологии
Язык: Python 3.x
Распознавание текста: easyocr
Обработка изображений: opencv-python (cv2)
Массивы данных: numpy

                                                      Установка и запуск

1. Установка зависимостей
Для запуска скрипта необходимо установить следующие библиотеки:

pip install easyocr opencv-python numpy


2. Запуск
Поместите файл ocr_tool_original.py в нужную директорию.
Сохраните изображение, с которого нужно извлечь данные, в ту же папку, назвав его Screenshot.jpg (или измените это имя в конце скрипта).
Запустите скрипт из терминала:

python ocr_tool_original.py

Скрипт поочередно попробует методы распознавания, а итоговый текст будет сохранен в файл result.txt или result_processed.txt.
